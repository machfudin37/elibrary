## E Library
E-Library merupakan aplikasi yang dibuat untuk staff perpustakaan dalam mengelola buku yang dipinjam kepada anggota perpustakaan. aplikasi ini dibangun menggunakan Framework Codeigniter versi 3.

```bash

```
